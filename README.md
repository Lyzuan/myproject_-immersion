# myproject_-immersion
Chatbot de atendimento para realizar apostas esportivas e auxiliar bancas que utilizam sites que geram códigos para cambistas. A intenção desse chatbot é ter um chat via whatsapp que funcionará 24 horas realizando atendimento. 
A intenção com esse projeto é integrar o site da empresa, conectar a api do aplicativo sa esportes e implementar uma plataforma para gerar um código de pagamento via pix.
aplicar o chatbot ao whatsapp e fazer com que o site leve o cliente diretamente ao atendimento que funcionará 24 horas dando assim vantagem a utilizacão de cambistas e gerar mais lucro sem a necessidade e pagamento por comissão.
